# Figure 1: Static vs. mobile targets on the 9/11 network

This directory contains the scripts and data required to reproduce 
**Figure 1** of the paper *Containment of Escaping Targets in 
Complex Networks*.

Figure 1 contrasts the isolation process on a fixed real-world 
topology (the 9/11 hijacker network) under two scenarios — 
stationary targets versus mobile targets — and quantifies how the 
target movement probability $p$ and the per-step removal budget 
$r$ jointly determine the containment cost.


## Files

### Scripts

| Script | Role |
|--------|------|
| `static_tia_vs_moving.py` | Generates the line-plot data (panels for static vs. random-walk targets on a synthetic Watts–Strogatz network). |
| `tia_pc_heatmap.py` | Generates the heatmap data (containment cost $P_C$ as a function of move probability $p$ and per-step removal budget $r$). |
| `draw_fig1.py` | Renders the final Figure 1 by combining the 9/11 network visualization (panels a–c) and the heatmap (panel d). Reads all CSVs from `data/`. |

### Data (`data/`)

| File | Description | Source |
|------|-------------|--------|
| `network.xml` | The 9/11 hijacker network in GraphML format. | Krebs (2002), *Connections* 24, 43–52 |
| `static_tia_fixed.csv` | Line-plot data under stationary targets. | Generated by `static_tia_vs_moving.py` |
| `static_tia_random_walk.csv` | Line-plot data under random-walk targets. | Generated by `static_tia_vs_moving.py` |
| `pc_random_ws.csv` | Heatmap data ($P_C$ over the $p$–$r$ grid). | Generated by `tia_pc_heatmap.py` |


## Reproducing the figure

The repository ships with the pre-computed intermediate CSVs in 
`data/`, so the figure can be reproduced directly:

```bash
python draw_fig1.py
```

This will produce `fig1.pdf` and `fig1.png` in this directory.

### Regenerating the data from scratch

To regenerate the intermediate CSVs from scratch, run the two 
data-generating scripts first, then render the figure:

```bash
python static_tia_vs_moving.py
python tia_pc_heatmap.py
python draw_fig1.py
```


## Configuration

Key parameters can be adjusted at the top of each script:

- `static_tia_vs_moving.py`:
  - `N = 1000`, `M = 5` — Watts–Strogatz network size and 
    half-degree
  - `TARGET_RATIO = 0.05`, `TARGET_MODE = 'random'` — target set 
    specification
  - `MOVE_PROB = 0.05`, `N_RUNS = 100` — target movement 
    probability and number of independent simulations
- `tia_pc_heatmap.py`:
  - `MOVE_PROBS` and `ATTACK_PER_STEPS` — the sweep grid
- `draw_fig1.py`:
  - `LAYOUT_SEED = 7` — controls the layout of the 9/11 network 
    visualization
