# Figure 4: Scale generalization and robustness to unseen target movement

This directory contains the scripts and data required to reproduce 
**Figure 4** of the paper *Containment of Escaping Targets in 
Complex Networks*.

Figure 4 combines two complementary generalization tests:

- **Top rows (a)–(h):** scale generalization. Holding the evasion 
  factor at $\lambda = 20$, the four methods are evaluated on 
  network sizes $N \in \{2^5, 2^6, \ldots, 2^{14}\}$, with the 
  largest tested instance ($N = 16{,}384$) exceeding the training 
  range $N \in [50, 100]$ by more than two orders of magnitude.
- **Bottom row (i)–(j):** robustness to unseen target movement 
  dynamics. Three movement regimes are compared on $N = 1{,}024$: 
  the unbiased random walk used at training time, a degree-biased 
  walk, and an outward-biased (start-point-biased) walk.

A single rendering script (`draw_node_scale.py`) reads four CSVs 
from `data/` and produces the full figure.


## Files

### Scripts

| Script | Role |
|--------|------|
| `comparison_method_node_scale.py` | Generates the scale-generalization data (panels a–h). Sweeps network size $N$ from $32$ to $16{,}384$ on all four topology–target regimes, at fixed $\lambda = 20$. Imports utility functions from `../fig3/comparison_method.py`. |
| `comparison_robustness.py` | Generates the degree-biased robustness data (bar chart in panels i–j). Sweeps the bias parameter $\tau$ across a wide range on $N = 1{,}024$ networks, under a degree-biased random walk. |
| `comparison_robustness_startpoint.py` | Generates the outward-biased robustness data (bar chart in panels i–j). Same sweep as above, but under a start-point-distance-biased random walk. |
| `draw_node_scale.py` | Renders the final Figure 4 by combining the four CSVs. |

### Data (`data/`)

| File | Description | Source |
|------|-------------|--------|
| `results_node_scale_tr5_lambda20.0_final.csv` | $P_C$ and ANC vs. $N$ for the four regimes, at $\lambda = 20$. | Generated by `comparison_method_node_scale.py` |
| `results_n1024.csv` | $P_C$ and ANC at $N = 1{,}024$ under the **unbiased random walk**. Used as the in-distribution reference in the bar chart. **This file is the same one produced by `../fig3/comparison_method.py`**; a snapshot is included here so this directory is self-contained. | Shared with `fig3/`; copy from `../fig3/data/results_n1024.csv` |
| `robustness_n1024.csv` | $P_C$ and ANC at $N = 1{,}024$ under the **degree-biased** random walk, over a $\tau$ sweep. | Generated by `comparison_robustness.py` |
| `robustness_startpoint_n1024.csv` | $P_C$ and ANC at $N = 1{,}024$ under the **outward-biased** (start-point-distance-biased) random walk, over a $\tau$ sweep. | Generated by `comparison_robustness_startpoint.py` |

preserved.


## Reproducing the figure

The repository ships with the pre-computed intermediate CSVs in 
`data/`, so the figure can be reproduced directly:

```bash
python draw_node_scale.py
```

This will produce `draw_node_scale_extended.pdf` and `.png` in 
this directory.

### Regenerating the data from scratch

The scale-generalization experiment is the most expensive of 
the three (dominated by the $N = 16{,}384$ instances):

```bash
python comparison_method_node_scale.py
```

This produces `data/results_node_scale_tr5_lambda20.0_final.csv`.

The two robustness sweeps can be regenerated with:

```bash
python comparison_robustness.py
python comparison_robustness_startpoint.py
```

These produce `data/robustness_n1024.csv` and 
`data/robustness_startpoint_n1024.csv` respectively.

The fourth file, `data/results_n1024.csv`, is shared with Fig. 3. 
If you have already generated it there, simply copy it over:

```bash
cp ../fig3/data/results_n1024.csv data/
```

Otherwise, generate it via `../fig3/comparison_method.py`.

Once all four CSVs are in place:

```bash
python draw_node_scale.py
```


## Configuration

Key parameters can be adjusted at the top of each script:

- `comparison_method_node_scale.py`:
  - `N_NODES_DEFAULT = [2**i for i in range(5, 15)]` — node-size grid
  - `TARGET_RATIO = 0.05`, `LAMBDA_EFF = 20.0` — fixed task settings
  - `SIMULATION_TIMES = 100` — independent simulations per condition
  - `N_WORKERS`, `N_WORKERS_RL` — multiprocessing pool sizes
- `comparison_robustness.py` / `comparison_robustness_startpoint.py`:
  - `N_NODES = 1024`, `TARGET_RATIO = 0.05`
  - `MOVE_PROB = 1.0`, `ATTACK_RATIO = 0.1` (corresponding to 
    $\lambda = 10$)
  - `TAU_VALUES` — the bias-strength sweep grid
- `draw_node_scale.py`:
  - `BAR_GRAPH_TYPE`, `BAR_DIST` — which of the four regimes the 
    bar chart should display (default `WS-random`)
  - `BAR_LAMBDA = 10`, `BAR_TAU = 1` — slice selectors for the 
    bar chart
