# Figure 3: Comparison with adaptive heuristic baselines on synthetic networks

This directory contains the scripts and data required to reproduce 
**Figure 3** of the paper *Containment of Escaping Targets in 
Complex Networks*.

Figure 3 compares MobileIsolator against three adaptive heuristic 
baselines (Adaptive TD, Adaptive T-Katz, Adaptive TIA) on the four 
synthetic topology–target regimes: Barabási–Albert random 
(heterogeneous-random), Barabási–Albert localized 
(heterogeneous-localized), Watts–Strogatz random 
(homogeneous-random), and Watts–Strogatz localized 
(homogeneous-localized).


## Files

### Scripts (run in numerical order)

| Script | Role |
|--------|------|
| `01_comparison_method.py` | Runs the full quantitative comparison across all four regimes and the 11 evasion-factor values $\lambda \in \{1, 2, 4, \ldots, 20\}$. Produces aggregate $P_C$ and ANC statistics over $100$ independent simulations per condition. |
| `02_generate_single_graph.py` | Generates one representative network instance per regime, runs each of the four methods on it, and records the removed-node sequence and final target positions used by the network panels (a)–(d). |
| `03_draw_fig3.py` | Renders the final Figure 3, combining the network panels (a)–(d) with the line plots (e)–(h) for $P_C$ and ANC. Reads all data from `data/`. |

### Data (`data/`)

| File / directory | Description | Source |
|------------------|-------------|--------|
| `results_n1024.csv` | Aggregate $P_C$ and ANC statistics (means and SEMs) over the 100 simulations per condition. | Generated by `01_comparison_method.py` |
| `single_graph/{config}/graph.graphml` | Representative network instance for each regime, used in network panels (a)–(d). | Generated by `02_generate_single_graph.py` |
| `single_graph/{config}/removed_{method}.txt` | Sequence of nodes removed by each method, one ID per line in removal order. | Generated by `02_generate_single_graph.py` |
| `single_graph/{config}/final_targets_{method}.txt` | Final target positions for each method at the end of its episode. | Generated by `02_generate_single_graph.py` |
| `single_graph/{config}/summary.json` | LCC size and other instance-level metadata. | Generated by `02_generate_single_graph.py` |

The four regime keys `{config}` are: `BA-random`, `BA-localized`, 
`WS-random`, `WS-localized`. The four method keys `{method}` are: 
`Adaptive_TD`, `Adaptive_Katz`, `Adaptive_TIA`, `GNN_RL` (the 
internal name for MobileIsolator).


## Dependencies

`01_comparison_method.py` and `02_generate_single_graph.py` both 
import the MobileIsolator agent class:

```python
from ppo_gnn_moving_gat_dualstream import DualStreamAgent
```

The module `ppo_gnn_moving_gat_dualstream.py` defining this class 
is shipped in the same directory. Both scripts also load the 
pre-trained policy weights from the repository-level `models/` 
directory:

```
../../models/pretrained_model.pt
```

(adjust the relative path in the scripts if you move the directory).


## Reproducing the figure

The repository ships with the pre-computed intermediate data in 
`data/`, so the figure can be reproduced directly:

```bash
python 03_draw_fig3.py
```

This will produce `draw_fig3_final_{config}.pdf` and `.png` in 
this directory. By default the script renders one regime per 
invocation, selected via the variables `SELECTED_GRAPH_TYPE` and 
`SELECTED_TARGET_DIST` at the top of `03_draw_fig3.py`.

### Regenerating the data from scratch

To regenerate the aggregate statistics from scratch, run:

```bash
python 01_comparison_method.py
```

This produces `data/results_n1024.csv`.

To regenerate the representative-network panels:

```bash
python 02_generate_single_graph.py
```

This populates `data/single_graph/{config}/` with the GraphML, 
removed-node sequences, and final-target files for each regime.

Once both steps are complete:

```bash
python 03_draw_fig3.py
```


## Configuration

Key parameters can be adjusted at the top of each script:

- `01_comparison_method.py`:
  - `N_NODES = 1024` — synthetic network size
  - `TARGET_RATIO = 0.05` — fraction of nodes designated as targets
  - `LAMBDA_EFF_VALUES = [1] + list(range(2, 22, 2))` — evasion 
    factor grid
  - `SIMULATION_TIMES = 100` — independent simulations per 
    condition
  - `N_WORKERS = 31`, `N_WORKERS_RL = 31` — multiprocessing pool 
    sizes
- `03_draw_fig3.py`:
  - `SELECTED_GRAPH_TYPE` and `SELECTED_TARGET_DIST` — which of 
    the four regimes to render
